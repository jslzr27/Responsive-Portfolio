Movie Music Enthusiast (M²E)

Everyone likes music, and everyone likes movies, so this app was born as a means to bridge the two. There's a lot of apps that give you music recommendations based on the music you like, and others that give you movie suggestions based on your tastes. We also happened to enjoy movie soundtracks, in particular. So this created an interesting question: what if I wanted to get music based on the movies I liked, or vice versa? This app lets you do that.

This website was created with collaboration from other fellow developers and it uses HTML5, CSS3, Bootstrap, Firebase, jQuery and Javascript as the technologies that power the app.

Users can input the train's name, destination, frequency of travel and next arrival. The technologies used to power this app are HTML, CSS, Javascript, jQuery, Bootstrap, Moment.js, Firebase.

Crystals Game

This game gives you a random number at the start, and you have to match the number in order to win.

Each crystal has a unique number to it and whenever you click on any crystal it will add up to the sum of the total number. but here is the catch, you don't know what is the secret number for each crystal so you would have to try with each.

Discover the numbers and you will be able to win. But if your score is higher that the number to match you lose.

Liri-Bot Node App

LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data.

    LIRI uses the following commands:

  • Command One: node liri.js concert-this "artist/band name here". This will search the Bands in Town Artist Events API for an artist and render information about each event to the terminal.

  • Command Two: node liri.js spotify-this-song "Song Name Here". This will search the Spotify API and give you back information about the song you provided

  • Command Three: node liri.js movie-this "movie name here". This will search the OMDB API and output information about the movie you typed in.

  • Command Four: node liri.js do-what-it-says. This will output the command placed in random.txt file
